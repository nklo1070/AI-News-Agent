# AI-News-Agent
AI News Agent

This pack is designed for a daily international news workflow in AI agent desktop or scheduled tasks.

## Files
- `core_system_prompt.txt`
- `source_list.json`
- `news_output_template.md`
- `macro_watchlist.json`
- `country_watchlist.json`
- `industry_watchlist.json`
- `company_watchlist.json`
- `task_prompt_abacus_desktop.txt`
- `README_abacus_desktop_setup.md`

## Recommended setup in Abacus Desktop
1. Upload all files in one chat.
2. Create a scheduled task.
3. Choose your preferred model.
4. Paste the contents of `task_prompt_abacus_desktop.txt` into the task prompt box.
5. Save and run daily.

## Good practice
- Keep source names identical to final links.
- Do not force every watchlist into every story.
- If source evidence is thin, explicitly say information is insufficient.
